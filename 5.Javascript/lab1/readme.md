### Lab 1 การตั้งชื่อตัวแปรที่ดี (Lab)

- ให้ประกาศตัวแปรชื่อ human และ name
- ใส่ชื่อตัวเองลงในตัวแปร name
- นำค่าที่อยู่ในตัวแปร name ไปใส่ให้ human
- เมื่อ console.log(human) ออกมาต้องเป็นชื่อตัวเอง

```
    let human, name;

    name = "Chayanin";

    human = name;

    console.log(human);
```
